# MatDash React Tailwind Components

## Overview
MatDash is a Material Design inspired dashboard template built with React and Tailwind CSS.

## Component Categories
- Dashboard layouts
- Analytics widgets
- Data visualization
- Admin panels
- User management
- Settings pages

## Features
- Material Design principles
- React 18
- Tailwind CSS
- Responsive dashboard
- Chart integration

## Usage
Components are located in `matdash-react-tailwind-free-main/`
