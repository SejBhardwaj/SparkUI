# HyperUI Component Collection

## Overview
HyperUI is a collection of free Tailwind CSS components for marketing and ecommerce websites.

## Component Categories
- Marketing sections
- Ecommerce components
- Application UI
- Navigation
- Forms
- Buttons

## Features
- Copy-paste ready
- Tailwind CSS v3
- Responsive design
- Accessible components

## Usage
Components are located in `hyperui-main/`
