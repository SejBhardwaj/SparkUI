# Shadcraft Free Components

## Overview
Shadcraft offers a collection of free, modern UI components with shadcn/ui styling.

## Component Types
- Buttons and inputs
- Cards and containers
- Navigation menus
- Modals and dialogs
- Data tables
- Charts

## Features
- shadcn/ui compatible
- Radix UI primitives
- Tailwind CSS styling
- Fully accessible
- Customizable

## Usage
Components are located in `shadcraft-free-main/`
