# Tailus HTML Components

## Overview
Tailus provides beautiful HTML components with Tailwind CSS styling.

## Component Categories
- Hero sections
- Feature sections
- Pricing tables
- Testimonials
- Call-to-action blocks
- Footer sections

## Features
- Pure HTML/CSS
- Tailwind CSS based
- No JavaScript required
- SEO friendly
- Fast loading

## Usage
Components are located in `html.tailus.io-main/`
