# Material Kit React Components

## Overview
Material Kit React is a comprehensive UI kit based on Material Design principles for React applications.

## Component Categories
- Material buttons
- Material cards
- Material forms
- Navigation drawers
- App bars
- Snackbars and dialogs

## Features
- Material Design 3
- React 18
- TypeScript support
- Responsive components
- Theme customization
- Icon library included

## Usage
Components are located in `material-kit-react-main/`

## Material Design Principles

Following Material Design guidelines:
- Material surfaces
- Elevation system
- Motion and animation
- Typography scale
- Color system

## Example

```jsx
import { Button, Card } from '@material-kit/react';

<Card elevation={2}>
  <Card.Content>
    <Button variant="contained" color="primary">
      Material Button
    </Button>
  </Card.Content>
</Card>
```
