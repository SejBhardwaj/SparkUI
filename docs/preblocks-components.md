# Preblocks Component Documentation

## Overview
Preblocks provides pre-built, customizable blocks for rapid website development.

## Block Categories
- Hero sections
- Feature grids
- Pricing sections
- Team sections
- Contact forms
- Blog layouts

## Features
- Ready-to-use blocks
- Easily customizable
- Responsive design
- Modern aesthetics
- Copy-paste friendly

## Usage
Components are located in `preblocks-main/`
