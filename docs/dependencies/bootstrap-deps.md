# Bootstrap UI Dependencies Audit

## Overview
Audit of dependencies used in Bootstrap UI library.

## Core Dependencies
- Bootstrap: ^5.3.0
- jQuery (optional for some components)
- Popper.js: ^2.11.0

## JavaScript Plugins
- Modal
- Dropdown
- Tooltip
- Popover

## Build Dependencies
- Sass
- Webpack

## Migration Notes
Bootstrap 5 removed jQuery dependency for most components.
