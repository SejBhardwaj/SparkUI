# Material Kit React Dependencies Audit

## Overview
Audit of dependencies used in Material Kit React library.

## Core Dependencies
- React: ^18.x
- Material-UI: ^5.x
- Emotion: ^11.x

## Additional Libraries
- React Router
- Prop Types
- Clsx

## Peer Dependencies
- React DOM
- React Scripts

## Bundle Size
Material-UI adds significant bundle size, consider tree-shaking.
