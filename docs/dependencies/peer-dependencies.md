# Peer Dependencies Documentation

## Overview
Documentation of peer dependency requirements across libraries.

## Common Peer Dependencies
- React: ^18.0.0
- React DOM: ^18.0.0
- TypeScript: ^5.0.0

## Version Compatibility
Ensure peer dependencies are compatible across all packages.

## Installation
```bash
npm install --legacy-peer-deps
# or
npm install --force
```

## Best Practices
- Specify peer dependencies in package.json
- Document version requirements
- Test with different peer versions
