# Input Component

## Overview
Flexible input component with validation, icons, and multiple states.

## Usage
```tsx
<Input label="Username" placeholder="Enter username" />
```

## Props
- label, error, helperText, fullWidth, disabled

## Features
- Multiple variants (text, email, password, number)
- Size options (xs, sm, md, lg, xl)
- Icon support (left/right)
- Validation
- Accessibility
