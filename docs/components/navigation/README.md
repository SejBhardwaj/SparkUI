# Navigation Components

## Overview
Complete navigation component library for building modern web applications.

## Components

### Navbar
Flexible navigation bar with variants, dropdown menus, and mobile support.
- Fixed, sticky, and transparent variants
- Responsive mobile menu
- Dropdown support
- Search functionality

### Breadcrumb
Breadcrumb navigation for showing current location in hierarchy.
- Multiple separator styles
- Collapse for long paths
- Full accessibility

### Tabs
Tab navigation for organizing content into sections.
- Horizontal and vertical orientations
- Multiple variants (underline, pills, enclosed)
- Keyboard navigation

### Accordion
Collapsible accordion for organizing content.
- Single or multiple expand modes
- Smooth animations
- Custom icons

### Pagination
Pagination for navigating through pages of content.
- Customizable page ranges
- First/last and prev/next controls
- Info display

### Sidebar
Sidebar navigation for app layouts.
- Fixed, overlay, and push variants
- Collapsible with animation
- Item groups

### Menu
Dropdown and context menus.
- Dropdown menus
- Context menus (right-click)
- Nested submenus
- Keyboard navigation

### Stepper
Multi-step navigation for wizards and processes.
- Horizontal and vertical orientations
- Step status indicators
- Navigation controls
