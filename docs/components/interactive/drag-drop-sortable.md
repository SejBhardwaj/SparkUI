# Sortable List Component

## Overview
Sortable list component with drag and drop functionality from Bootstrap UI.

## Features
- Drag and drop reordering
- Touch support
- Keyboard navigation
- Accessibility features

## Usage
```jsx
<SortableList items={items} onReorder={handleReorder} />
```

## Props
- `items`: Array of items to display
- `onReorder`: Callback when items are reordered
- `disabled`: Disable sorting
- `handle`: Custom drag handle
