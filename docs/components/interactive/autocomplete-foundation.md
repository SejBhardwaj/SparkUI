# Autocomplete Foundation

## Overview
Base autocomplete component from TailAdmin with essential functionality.

## Features
- Real-time filtering
- Dropdown suggestions
- Keyboard navigation
- Loading states
- Error handling

## Usage
```jsx
<Autocomplete
  options={options}
  onSelect={handleSelect}
  placeholder="Search..."
/>
```

## Props
- `options`: Array of suggestions
- `onSelect`: Callback when option selected
- `loading`: Show loading state
- `error`: Error message
- `placeholder`: Input placeholder

## Implementation
See TailAdmin form elements for details.
