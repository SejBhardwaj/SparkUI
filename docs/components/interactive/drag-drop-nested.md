# Nested Drag and Drop Component

## Overview
Nested drag and drop component from Shadcraft for hierarchical data structures.

## Features
- Multi-level nesting
- Drag between levels
- Visual feedback
- Drop zones
- Collision detection

## Usage
```jsx
<NestedDragDrop tree={treeData} onUpdate={handleUpdate} />
```

## Props
- `tree`: Hierarchical data structure
- `onUpdate`: Callback when structure changes
- `maxDepth`: Maximum nesting level
- `collapsible`: Allow collapsing nodes
