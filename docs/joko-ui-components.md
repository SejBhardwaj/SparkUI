# Joko UI Component Library

## Overview
Joko UI provides modern, accessible React components with a focus on developer experience.

## Component Categories
- Layout components
- Form elements
- Data display
- Feedback components
- Navigation
- Overlays

## Features
- TypeScript support
- Fully accessible (WCAG 2.1)
- Customizable theming
- Responsive design
- Tree-shakeable

## Usage
Components are located in `joko-ui-main/`
