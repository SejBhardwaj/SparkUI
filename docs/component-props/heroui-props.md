# HeroUI Component Props

## Overview
Documentation of component props from HeroUI library.

## Component System
- Modern React components
- TypeScript support
- Accessibility built-in

## Props Interface
- variant: string
- color: string
- size: string
- isDisabled: boolean
- isLoading: boolean
- radius: 'none' | 'sm' | 'md' | 'lg' | 'full'

## Customization
Theme tokens and CSS variables.
