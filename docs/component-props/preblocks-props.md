# Preblocks Component Props

## Overview
Documentation of component props from Preblocks library.

## Block Components
- Hero sections
- Feature blocks
- Testimonials
- Call-to-action sections

## Props Interface
- layout: string
- theme: 'light' | 'dark'
- spacing: string
- alignment: 'left' | 'center' | 'right'

## Customization
Flexible layout and styling options.
