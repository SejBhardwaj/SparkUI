# TailAdmin Button Component Props

## Overview
Documentation of button component props from TailAdmin library.

## Props Interface
- variant: 'primary' | 'secondary' | 'outline' | 'ghost'
- size: 'sm' | 'md' | 'lg' | 'xl'
- disabled: boolean
- loading: boolean
- icon: ReactNode
- fullWidth: boolean
