# Bootstrap Button Component Props

## Overview
Documentation of button component props from Bootstrap UI library.

## Props Interface
- variant: 'primary' | 'secondary' | 'success' | 'danger' | 'warning' | 'info' | 'light' | 'dark'
- size: 'sm' | 'lg'
- outline: boolean
- disabled: boolean
- active: boolean
- block: boolean

## Bootstrap Classes
btn, btn-primary, btn-outline-*, btn-sm, btn-lg
