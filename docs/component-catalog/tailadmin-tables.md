# TailAdmin Table Components Catalog

## Table Variants
- Basic data table
- Striped rows table
- Bordered table
- Hoverable rows
- Compact table

## Features
- Sortable columns
- Filterable data
- Pagination controls
- Row selection (single/multiple)
- Expandable rows
- Fixed headers
- Responsive tables

## Column Types
- Text columns
- Number columns
- Date columns
- Action columns
- Status badge columns
- Avatar columns

## Props
- `data`: Array of row objects
- `columns`: Column definitions
- `sortable`: boolean
- `selectable`: boolean
- `pagination`: boolean
- `onRowClick`: (row) => void
