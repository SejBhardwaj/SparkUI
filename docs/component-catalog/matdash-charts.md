# MatDash Chart Components

## Chart Types
- Line charts
- Bar charts (vertical/horizontal)
- Pie and donut charts
- Area charts
- Scatter plots
- Mixed charts

## Chart Features
- Interactive tooltips
- Legend customization
- Axis configuration
- Data labels
- Zoom and pan
- Export functionality

## Integration
- Chart.js based
- Responsive sizing
- Theme support
- Animation options

## Use Cases
- Sales analytics
- User metrics
- Performance monitoring
- Financial data
