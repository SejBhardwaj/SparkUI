# NamelessUI Component Collection

## Core Components
- Button system
- Input fields
- Select dropdowns
- Checkboxes and radios
- Toggle switches
- Modals
- Tooltips
- Popovers
- Dropdowns

## Advanced Components
- Date pickers
- Color pickers
- File uploaders
- Rich text editors
- Code editors

## Features
- Headless UI approach
- Unstyled by default
- Full customization
- Accessibility built-in
- TypeScript support
