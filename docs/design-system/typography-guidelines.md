# Typography Usage Guidelines

## Font Families
- **Sans-serif**: Use for body text, UI elements, and most content
- **Serif**: Use for long-form reading, editorial content
- **Monospace**: Use for code, technical content, data

## Font Sizes
### Headings
- H1: 3xl - 5xl (30px - 48px)
- H2: 2xl - 4xl (24px - 36px)
- H3: xl - 3xl (20px - 30px)
- H4: lg - 2xl (18px - 24px)
- H5: base - xl (16px - 20px)
- H6: sm - lg (14px - 18px)

### Body Text
- Large: lg (18px)
- Default: base (16px)
- Small: sm (14px)
- Extra small: xs (12px)

## Font Weights
- Headings: semibold (600) or bold (700)
- Body text: normal (400)
- Emphasis: medium (500) or semibold (600)
- De-emphasis: light (300)

## Line Heights
- Headings: tight (1.25) to snug (1.375)
- Body text: normal (1.5) to relaxed (1.625)
- UI elements: tight (1.25) to normal (1.5)

## Best Practices
1. Maintain consistent hierarchy
2. Use appropriate contrast ratios
3. Limit font variations
4. Consider readability
5. Test on different devices
