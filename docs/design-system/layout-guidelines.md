# Layout System Guidelines

## Grid System
- Use 12-column grid for complex layouts
- Use 4-6 columns for simpler layouts
- Maintain consistent gutters (16px or 24px)

## Containers
- sm (640px): Mobile landscape
- md (768px): Tablet
- lg (1024px): Desktop
- xl (1280px): Large desktop
- 2xl (1536px): Extra large screens

## Spacing
- Use consistent spacing scale
- Prefer spacing tokens over arbitrary values
- Maintain vertical rhythm

## Flexbox
- Use for one-dimensional layouts
- Prefer for navigation, toolbars
- Good for centering content

## Grid
- Use for two-dimensional layouts
- Prefer for card grids, galleries
- Better for complex layouts

## Best Practices
1. Mobile-first approach
2. Responsive breakpoints
3. Consistent spacing
4. Semantic HTML
5. Accessibility considerations
