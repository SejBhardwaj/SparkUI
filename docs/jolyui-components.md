# JolyUI Component Library

## Overview
JolyUI is a modern, joyful UI component library with delightful animations and interactions.

## Component Categories
- Interactive buttons
- Animated cards
- Smooth transitions
- Micro-interactions
- Loading states
- Toast notifications

## Features
- Delightful animations
- Smooth interactions
- Modern design
- Lightweight
- Easy to customize
- Performance optimized

## Usage
Components are located in `jolyui-main/`

## Design Philosophy

JolyUI focuses on creating joyful user experiences through:
- Subtle animations
- Smooth transitions
- Playful interactions
- Attention to detail
