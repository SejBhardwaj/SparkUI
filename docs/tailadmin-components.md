# TailAdmin Dashboard Template Components

## Overview
TailAdmin is a free Tailwind CSS dashboard template with modern components.

## Key Components
- Dashboard layouts
- Charts and graphs
- Tables
- Forms
- UI elements
- Authentication pages

## Features
- Fully responsive
- Dark mode support
- Tailwind CSS based
- Modern design patterns

## Usage
Components are located in `tailadmin-free-tailwind-dashboard-template-main/`
