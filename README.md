# UI Components Collection

A comprehensive collection of modern UI component libraries and templates for building beautiful web applications.

## Overview

This project aggregates and documents multiple UI component libraries, providing a unified resource for developers to explore, compare, and utilize various design systems and component collections.

## Included Libraries

- **TailAdmin** - Free Tailwind CSS dashboard template
- **HyperUI** - Collection of free Tailwind CSS components
- **Bootstrap UI Components** - Bootstrap-based UI components
- **Joko UI** - Modern UI component library
- **Dev.UI** - Developer-focused UI components
- **Tailus** - HTML components with Tailwind CSS
- **Shadcraft** - Free component collection
- **MatDash** - React + Tailwind dashboard template
- **Preblocks** - Pre-built component blocks
- **Forklifter** - UI component toolkit
- **Shadcn UI Angular** - Angular port of Shadcn UI
- **JolyUI** - Elegant UI components
- **NamelessUI** - Minimalist component library
- **Material Kit React** - Material Design React components
- **HeroUI** - Next-generation UI components

## Project Structure

```
.
├── docs/                    # Documentation for each library
├── src/                     # Source code
│   ├── components/         # Unified component implementations
│   ├── hooks/              # React hooks
│   ├── utils/              # Utility functions
│   ├── types/              # TypeScript type definitions
│   └── design-tokens/      # Design system tokens
├── [library-folders]/      # Individual library sources
└── PROJECT_PLAN.md         # Development roadmap
```
